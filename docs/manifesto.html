AIbasic v1.0 represents a paradigm shift: from a simple execution VM to a complete, production-ready infrastructure with integrated data services, security, messaging, and cloud storage.

7) Advanced Control Flow

Beyond simple sequential execution:

Jumps (GOTO)

10 set counter to 0
20 print counter
30 increment counter by 1
40 if counter < 10 jump to line 20
50 print "Done"

Error Handling (ON ERROR GOTO)

10 on error goto 100
20 (postgres) query "SELECT * FROM missing_table"
30 goto 200

100 print "Database error:" and _last_error
110 print "Error at line:" and _last_error_line

200 print "Complete"

Subroutines (CALL/RETURN)

10 call 1000
20 print "Result:" and result
30 goto 999

1000 set x to 10
1010 set y to 5
1020 add x to y and store in result
1030 return

999 print "End"
