Semantics

Intent Mapping

Each instruction becomes a node in an intent graph. The compiler chooses strategies based on context (e.g., “Create Excel report” → direct writer if available, else RPA through Excel).

Data Operations

Load "sales.csv" as Sales.
Filter Sales where Country == "Italy" into It.
Group It by "Month" and compute sum of "Revenue" as Total into MoM.
Save MoM into the data service "Analytics".

RPA Operations

Open Excel.
Create a worksheet "Monthly".
Fill cells A1:D1 with ["Month","Country","Sales","Profit"].
Paste data from table MoM.
Save as "monthly.xlsx".

Control Flow

LABEL Retry: Click the "Login" button.
Wait for text "Welcome" up to 6 seconds.
If not Found("Welcome") then Go to Retry.

Errors & Ambiguity

  • Column ambiguity → quote names.
  • Selector fragility → prefer roles/ARIA/text; provide timeouts and retries.
  • Type uncertainty → compiler inserts safe casts and validations.
