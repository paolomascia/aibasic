Language — Natural Instructions with Line Numbers

Human-readable, English sentences with optional line numbers keep scripts auditable and easy to diff. The compiler resolves ambiguity using the AI Context and emits deterministic plans.

Instruction Style

  • Imperative: Load..., Filter..., Group..., Open..., Click...
  • Quote ambiguous fields: "Order Date", "Revenue"
  • Labels and control: LABEL Retry:, Go to Retry, Repeat ... until ...

Variables as Semantic Entities

Variables carry schema, provenance, and mutability. Names are TitleCase in instructions → snake_case in generated code.
