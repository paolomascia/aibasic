Language

Purpose

AIbasic is a text-first language with English, numbered instructions, focused on data manipulation (CSV/Parquet/DB) and RPA (browser/desktop). The compiler concatenates Python code generated for each line.

Program structure

<line_number> <instruction in English>
  • line_number: positive integer (commonly 10, 20, 30…).
  • Each line defines/transforms data or automates actions.
  • The compiler sorts by line number; labels and line refs drive control flow.

Comments & Labels

  • Inline comment: # ...
  • Line comment: a line starting with #
  • Label: LABEL Name: before the instruction text.
30 LABEL RetryStep: Click the "Login" button.

Lexicon & style

  • Imperative/declarative style: “Load…”, “Filter…”, “Group… by… and compute…”, “Click…”.
  • Variables in TitleCase → snake_case in Python.
  • Ambiguous fields quoted: "Order Date", "Revenue".
  • Literals: strings, numbers, dates (YYYY-MM-DD), relative time (today, last month).
  • Paths: quoted strings.
  • RPA selectors: visible text, CSS/XPath, roles/ARIA.

Types & data structures

  • Scalars: int, float, bool, string, date, datetime
  • Collections: list[T], dict[K,V]
  • Table/DataFrame, Record/Row, Selector, Handle

AI Context

The AI Context is an incremental JSON describing metadata for each variable (schema, shape, derives_from, privacy, mutability, …) and global state (symbol_table, imports, resources, capabilities).
