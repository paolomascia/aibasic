Example 1 — Data Filtering and Aggregation

AIbasic Code

10 Load "sales.csv" as Sales.
20 Filter Sales where "Country" == "Italy" into ItSales.
30 Group ItSales by "Year" and compute sum of "Revenue" as Total into YearlyRev.
40 Export YearlyRev to CSV at "yearly_revenue.csv".
50 Print "Done."

Compiled Python

import pandas as pd

# 10
sales = pd.read_csv("sales.csv")

# 20
itsales = sales.loc[sales["Country"] == "Italy"].copy()

# 30
yearlyrev = itsales.groupby("Year")["Revenue"].sum().reset_index(name="Total")

# 40
yearlyrev.to_csv("yearly_revenue.csv", index=False)

# 50
print("Done.")

Example 2 — Browser Automation

AIbasic Code

10 Open a new browser as Browser.
20 Open "https://example.com/login" in Browser as Page.
30 Type "user@example.com" into css=#email on Page.
40 Type "secret" into css=#password on Page.
50 Click the "Login" button on Page.
60 Wait for text "Welcome" on Page up to 10 seconds.
70 Read text from css=.balance on Page into BalanceText.
80 Print BalanceText.

Compiled Python

from playwright.sync_api import sync_playwright

# 10-20
_pw = sync_playwright().start()
_browser = _pw.chromium.launch(headless=True)
page = _browser.new_page()
page.goto("https://example.com/login")

# 30-50
page.fill("css=#email", "user@example.com")
page.fill("css=#password", "secret")
page.get_by_role("button", name="Login").click()

# 60
page.wait_for_selector("text=Welcome", timeout=10000)

# 70
balance_text = page.text_content("css=.balance")

# 80
print(balance_text)

Example 3 — Combined Data + RPA Workflow

AIbasic Code

10 Load "orders.csv" as Orders.
20 Filter Orders where "Status" == "Paid" and "Country" == "Italy" into PaidIt.
30 Group PaidIt by "Month" and compute sum of "Amount" as Total into MonthlyRev.
40 Export MonthlyRev to CSV at "monthly_it_revenue.csv".

50 Open a new browser as Browser.
60 Open "https://portal.example.com" in Browser as Portal.
70 Type "ops@example.com" into css=#user on Portal.
80 Type "pass123" into css=#pass on Portal.
90 Click the "Login" button on Portal.
100 Wait for text "Dashboard" on Portal up to 10 seconds.
110 Upload file "monthly_it_revenue.csv" into css=#uploader on Portal.
120 Click the "Submit" button on Portal.
130 Wait for text "Upload completed" on Portal up to 20 seconds.
140 Print "Automation complete."

Compiled Python

import pandas as pd
from playwright.sync_api import sync_playwright

# 10
orders = pd.read_csv("orders.csv")

# 20
paidit = orders[(orders["Status"] == "Paid") & (orders["Country"] == "Italy")].copy()

# 30
monthlyrev = paidit.groupby("Month")["Amount"].sum().reset_index(name="Total")

# 40
monthlyrev.to_csv("monthly_it_revenue.csv", index=False)

# 50-60
_pw = sync_playwright().start()
_browser = _pw.chromium.launch(headless=True)
portal = _browser.new_page()
portal.goto("https://portal.example.com")

# 70-120
portal.fill("css=#user", "ops@example.com")
portal.fill("css=#pass", "pass123")
portal.get_by_role("button", name="Login").click()
portal.wait_for_selector("text=Dashboard", timeout=10000)
portal.set_input_files("css=#uploader", "monthly_it_revenue.csv")
portal.get_by_role("button", name="Submit").click()
portal.wait_for_selector("text=Upload completed", timeout=20000)

# 140
print("Automation complete.")

Example 4 — Loop and Condition

AIbasic Code

10 Let Threshold be 1000.
20 For each row in Sales do
30     If row."Revenue" > Threshold then Print row."CustomerID".
40 End for.

Compiled Python

threshold = 1000

for _, row in sales.iterrows():
    if row["Revenue"] > threshold:
        print(row["CustomerID"])
